The vision server from the 1360 is currently to be used until the one from the 6.1 branch has been tested.
It is not yet included in the installer and still has to be installed manually.

Installation:

0. D:\IntactVisionServer directory should be deleted.
1. copy files from this directory to D:\Tools.
   I.e. there are then the directories D:\Tools\IntactVisionServer and D:\Tools\IntactVisionServer.Scripting
2. run Install_VisionServer_WinService.bat as admin
3. as admin C:\windows\Microsoft.Net\Framework\v4.0.30319\regasm D:\Tools\IntactVisionServer.Scripting\IntactVisionServer.Scripting.dll /codebase 
   Execute it.
4. call up services and check whether IntactVisionServer is running.
5. restart
6. test with Intact.exe
